# Прогон ресёрча A0 — Таблицы Шульте (июль 2026, пилот)

> Первый прогон Контура A. Цели пилота: обкатать чек-лист скрининга
> (`../contour-a-publications.md` §3) и формат карточки-кандидата на живом
> материале. Исполнитель: ИИ-ассистент; валидация — Степан.

## Запросы и источники

Поиск: русскоязычные публикации о нормативных данных «Таблиц Шульте»
(эффективность работы, время на таблицу) на условно здоровых взрослых.
Запросы: «таблицы Шульте нормативные данные условно здоровые возрастные
группы», «Шульте эффективность работы нормативные показатели выборка M SD»,
«исследование внимание студентов таблицы Шульте выборка», «преддементные
состояния молодой возраст Шульте».

## Находки и скрининг

### ✅ Кандидат 1 (единственный прошедший, слабый)
- **Источник:** «Преддементные состояния у лиц молодого возраста», студенческая
  научная конференция СибАК (sibac.info/studconf/natur/xxxv/43164), ~2014,
  база — Нижегородская гос. медицинская академия.
- **Данные:** 100 респондентов молодого возраста; среднее время на таблицу
  30,3 с, SD ≈ 3,59 с.
- **Скрининг по чек-листу:** методика совпадает (время на таблицу) — да;
  n указан (100) — да; M±SD есть — да; возрастная ячейка — «молодой возраст»
  (точные границы уточнить по полному тексту); язык/культура — ru; год — 2014.
- **Слабости (честно):** студенческая конференция → тип источника «другое»
  (E = 6 баллов); стратификация только «молодой возраст»; часть выборки
  имела когнитивные жалобы — уточнить, считалось ли среднее по всем 100 или
  по здоровой подгруппе (**ключевой вопрос к валидации**).
- **Карточка-кандидата** — см. ниже.

### ❌ Отсеяны (зафиксировано, чтобы не искать повторно)
1. **Клинические рекомендации Минздрава «Когнитивные расстройства у лиц
   пожилого и старческого возраста», приложение Г12 (Таблицы Шульте)** —
   официальный статус высокий, но в открытом доступе (sudact.ru, 403 из
   песочницы) не удалось извлечь: нет размера ячейки и SD. **Достать полный
   текст** — если там есть пороги, это сильный cutoff-источник. → В список A1.
2. **Классика методичек** («30–50 с, чаще 40–42 с» — восходит к пособиям типа
   Рубинштейн С.Я.) — только диапазон, без n ячейки и SD → по чек-листу не
   проходит (критерии 2–3). Может жить как справочный cutoff, если Степан
   укажет издание и страницу.
3. Сайты-агрегаторы (pro-mozg, memo.expert, gb.ru и т.п.) — «норма 25–30 с»
   без выборки и источника → не кандидаты по определению.
4. КиберЛенинка «Исследование у студентов свойств внимания…» — по аннотации
   внимание мерили корректурной пробой Бурдона и Пьерона-Рузера, не Шульте →
   мимо методики (но потенциальный источник для корректурной пробы — в A1).

## Выводы пилота (важно для этапа A1)

1. **Чек-лист работает**: из ~10 найденных источников честно отсеялось всё,
   кроме одного слабого кандидата. Система скоринга дальше сама занизит балл —
   конвейер не пропускает мусор в базу.
2. **Открытый веб беден на полные нормативные таблицы**: у сильных источников
   (eLibrary, клин. рекомендации, диссертации) полные тексты за пейволом или
   недоступны из среды разработки (403). **Решение для A1:** Степан присылает
   PDF статей/методичек, ассистент оцифровывает таблицы в карточки — это ровно
   «Контур A» из ТЗ §7 (ручная/ИИ-оцифровка).
3. Замечена работа по **нормативным данным Trail Making Test для российской
   популяции** — методика-аналог Шульте (семейство). Для нашей карточки Шульте
   это `procedure_mismatch`, но для международной дорожки — прямой кандидат.
