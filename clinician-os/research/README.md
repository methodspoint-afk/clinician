# Журнал ресёрча публикаций (Контур A)

Каждый прогон мониторинга публикаций — отдельный файл `ГГГГ-ММ-методика.md`:
что искали (запросы, источники), что нашли, что отсеяли и почему (чек-лист —
`../contour-a-publications.md` §3), карточки-кандидаты (§4).

Кандидаты всегда рождаются черновиками; в базу норм попадают только после
валидации специалистом. Журнал не чистится — это история происхождения норм.
